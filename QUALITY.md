# 质量门（QUALITY）

> 本文件汇总 video-script-creator v3 的质量把控标准。
> **撰稿人自查 + 团队互审 + 专家终审** 三道关卡（源自魏桢公开访谈）。
> 详细 Checklist 见 [`quality-checklist.md`](./references/quality-checklist.md)。

---

## 1. 量化质量门（10 项可验证指标）

> 把"3 道递进有因果链"等主观标准，量化为可被模型自动验证的指标。

| # | 检查项 | 量化标准 | 验证方式 |
|---|--------|---------|---------|
| 1 | **钩子质量** | 必须从 30 个公式（`hook-formulas.md`）中选 1，并标注公式编号 | grep / 编号检查 |
| 2 | **因果链** | 每道递进 ≥3 个"因为-所以"或"导致 / 因此 / 于是"等因果关键词 | 词频统计 |
| 3 | **三连接** | 必须填满 3×N 表格（N=用户要求元素数），任一格为空 = 不通过 | 表格填空检查 |
| 4 | **金句密度** | 每 2 分钟 ≥1 句可截图金句（10 分钟视频 ≥5 句） | 段落计数 |
| 5 | **一句话锚定** | 长度 20-50 字，含动词 + 名词 + 情感锚 | 字符 + 词性检查 |
| 6 | **桢公子适配** | 长句（>50 字）占比 ≤30%，每分钟字数 200-250 | 句子长度统计 |
| 7 | **数据来源** | 所有数字带可点击链接，≥80% 来自权威源（官方/学术/行业） | 来源标注检查 |
| 8 | **人物出场** | ≥2 个真实人物故事（姓名 + 场景 + 行动） | 段落识别 |
| 9 | **情绪曲线** | 至少 2 高点 + 1 低点 + 结尾缓慢下降（不是高点） | Mermaid 图检查 |
| 10 | **回环升华** | 结尾 ≥1 个句子呼应开头的画面 / 数字 / 命题 | 关键词重叠检查 |

## 2. 一票否决项（出现任何一条，必须修改）

| # | 一票否决项 | 触发后动作 |
|---|-----------|-----------|
| 1 | **核心数据错误**（年份 / 海拔 / 距离 / 数量） | 修正 + 加来源 |
| 2 | **史实错误**（事件 / 人物 / 时间） | 修正 + 引用权威源 |
| 3 | **抄袭 / 洗稿**（未注明出处的原文复用超过 50 字） | 删除 + 重写 |
| 4 | **政治敏感**（涉政 / 涉民族 / 涉宗教 / 涉领土） | 整段删除或拒发 |
| 5 | **价值观偏离**（性别歧视 / 地域歧视 / 职业歧视等） | 修正或拒发 |
| 6 | **缺乏科学依据的伪科学** | 修正或加免责声明 |
| 7 | **立意不清晰**（讲完一遍不知道在讲什么） | 重写立意段 |
| 8 | **画面与文字脱节**（每分钟超过 3 句无画面） | 加画面钩子 |
| 9 | **商务擦边**（产品露出过于直接，违反"人文化包装"） | 调整露出度 |
| 10 | **钩子与正文无关** | 改钩子或改正文 |

---

## 3. 加分项（出现越多越好）

- 金句密度高于平均水平
- 类比陌生化精准（让人"哇"出来）
- 时间跨度和空间跨度都拉满
- 结尾升华与开头呼应
- 让读者产生"转发冲动"（态度大于信息）
- 文字可以脱离视频独立成稿（长期留存）
- **商务项目专项**：产品是"自然发现的"而非"被推销的"
- **风格专项**：3 道递进有清晰的因果链

## 4. 三道审校关

### 4.1 第一关：作者自审（撰稿人自查）

完成 Phase 5 之后立即执行，详见 [`quality-checklist.md §第一关`](./references/quality-checklist.md)。

### 4.2 第二关：交叉审核（团队互审）

模拟星球所内部"所长 + 桢公子 + 导演 + 动画师 + 地图师 + 素材编辑"6 视角审核。

| 视角 | 关注点 |
|------|--------|
| **所长**（耿华军） | 是否回答了"基本问题"？是否提出了"经典理念"？ |
| **桢公子**（魏桢） | 念出来是否流畅？每句话是否有画面？整体是否克制？ |
| **导演** | 画面钩子？节奏张弛？呼吸点？ |
| **动画师** | 抽象概念是否有可视化？数字是否有视觉表达？ |
| **地图师** | 是否需要新增地图？标注是否清晰？ |
| **素材编辑** | 素材缺口？是否需要补拍或 3D 替代？ |

**互审会输出三类反馈**：
1. **必须改**：影响命题 / 史实 / 节奏的问题
2. **建议改**：可优化但不影响主线的问题
3. **可保留**：暂时不改，看后续反馈

### 4.3 第三关：行业专家审（终审）

| 选题类型 | 推荐专家 |
|---------|---------|
| 自然地理 / 地质 | 中科院 / 高校地质 / 地理学者 |
| 历史人文 | 高校历史学者 / 博物馆研究员 |
| 工程 | 行业内部专家 / 设计院工程师 |
| 生态 / 生物 | 自然保护区研究员 / 高校生态学者 |
| 城市 | 城市规划学者 / 当地文史专家 |

专家审稿输出模板：

```
【专家审稿意见 - XX 主题】

史实性问题：
1. （如有，请具体指出，并提供权威来源）
2. ……

数据性问题：
1. ……

观点性问题：
1. ……

建议性意见（可改可不改）：
1. ……

总体评价：
A. 可发布
B. 需小修
C. 需大修
```

## 5. 各 Phase 完成标志（验收清单）

### Phase 1-2 完成标志

- [ ] ≥5 位创作者分析，具体技法级洞察（不止"风格好"）
- [ ] ≥15 个事实元素，有可追溯来源
- [ ] 创作者笔法库覆盖"定位 / 钩子 / 主线 / 金句 / 风险"5 维

### Phase 3-4 完成标志

- [ ] 3 个方案彼此清晰可区分（类型 / 钩子 / 主线都不同）
- [ ] 3 个方案各有 ≥3 个金句候选
- [ ] 对比矩阵含通用 + 商务专项 + 风格特色维度
- [ ] 个人推荐理由具体可执行（不止"我推荐 A"）

### Phase 5 完成标志

- [ ] 选定方案有五阶段情绪弧线（上扬 / 发展 / 下坠 / 回升 / 绵长）
- [ ] "三连接"规则满足（如连接差异元素）
- [ ] 锚定句可截图转发（20-50 字，含动词 + 情感锚）
- [ ] 生产指导包含拍摄 / 音乐 / 旁白 / 视觉
- [ ] Mermaid 叙事模型图清晰

### Phase 6 交付标志

- [ ] 所有来源带可点击链接引用
- [ ] 走 `html-report` 生成精美 HTML
- [ ] 可选：走 `docx` 生成 Word 版
- [ ] 三道审校关全部通过
- [ ] 一票否决项 0 条触发

## 6. 星球研究所的"长期主义"标准

> 魏桢原话：
> "我们创作的核心原则是解答基本问题，提出经典理念。让内容能够经得住时间的考验。"
> "我们距离百年老店，还有 90 年。"

### 终极自检问题

> 三年后、五年后、十年后再看这期视频，它还会让人想看第二遍吗？
>
> - **是** → 通过
> - **需更新数据** → 通过（数据是表的，理念是里的）
> - **热点过了就没意义了** → 重做

### "百年老店"标准推论

- **拒绝一次性热点**：今天的爆款，三年后是路人
- **优先"基本问题"**：什么是 X？X 从哪里来？X 意味着什么？
- **坚持"经典理念"**：可引用、可截图、可传家的句子
- **数据会过期，理念不会**：用数据支撑理念，不用理念解释数据
