# 创意原则（PRINCIPLES）

> 本文件汇总 video-script-creator v3 的核心创作原则。SKILL.md 是"流程"，本文件是"原则"。
> 撰稿过程中遇到"怎么写才地道"的问题，优先查本文件。

---

## 1. 事实优先（最高原则）

> **没有事实，就没有视频。** 每支脚本的每个论断必须可追溯到权威来源。

### 1.1 事实类型

| 类型 | 比例 | 范例 |
|------|------|------|
| **硬数据** | ≥40% | "福建 1400 座岛屿，比海南多" |
| **人物故事** | ≥20% | "校长林文清把学校变成避难所" |
| **地理 / 语境** | ≥20% | "八山一水一分田，亚热带季风" |
| **技术 / 领域** | ≥20% | "微电网 = 源-网-荷三要素" |

### 1.2 事实来源优先级

1. 政府官方（统计局 / 部委 / 地方政府公报）
2. 学术论文（中科院 / 高校 / 同行评审）
3. 行业报告（设计院 / 行业协会 / 头部研究机构）
4. 权威媒体（新华社 / 人民日报 / 央视 / 中国国家地理）
5. 头部创作者已发布作品（星球所 / 地球知识局等）

**不接受的来源**：百度百科、未署名自媒体、营销号。

### 1.3 数字呈现的 3 个铁律

- **必带单位**（km / 千米 / 公里 不能混用）
- **必带来源链接**（精确到文献 / 报告 / 官方）
- **必带类比**（"450 亿立方米 = 3000 个西湖"，陌生数字→熟悉类比）

## 2. 风格混搭（不是单一锚点）

> **选 1-2 位 UP 主作为主参考，星球所风格作为克制的底色。**

### 2.1 风格选择决策树

```
选题偏向...
├── 宏大地理 / 时间尺度大  → 星球所 + 中国国家地理
├── 城市故事 / 人物主线    → 星球所 + 小约翰可汗
├── 反常识 / 议题           → 地球知识局 + 半佛仙人
├── 科技产品 / 工程师视角   → 何同学 + 稚晖君
├── 法理 / 价值观           → 罗翔 + 老番茄
├── 历史彩蛋 / IP 借势      → 御史房 + 星球所
├── 自然博物                → 无穷小亮 + 汪品先
└── 工程奇迹 / 行业         → 星球所 + 影视飓风
```

### 2.2 风格混搭的边界

- **可以混搭**：钩子用 A，主线用 B，金句风格用 A
- **不要混搭**：克制的底色（星球所）+ 高度抒情的笔法——互相抵消
- **测试方法**：把方案念出来，如果觉得"油腻"或"突兀"，就是混搭失败

## 3. 三大时间尺度 + 三大空间维度（选题工具）

### 3.1 三大时间尺度（耿华军原话）

> "6500万年的华夏山水，1万年的文明史诗，100年的重塑山河。"

| 时间尺度 | 对应命题 | 范例 |
|---------|---------|------|
| **6500万年** | 自然地理、华夏山水 | 青藏高原 / 长江 / 三江并流 |
| **1万年** | 中华文明、人类活动 | 农耕起源 / 早期城市 / 文明兴衰 |
| **100年** | 重塑山河、现代工程 | 100年重塑山河 / 数字孪生水利 |

### 3.2 三大空间维度

| 维度 | 内容 |
|------|------|
| **自然地理** | 山川、河流、气候、生态 |
| **人文历史** | 文明、文化、历史、人物 |
| **现代工程** | 基础设施、产业、城市、科技 |

撰稿时优先选择 **1-2 个时间尺度 + 1-2 个空间维度** 作为主线。

## 4. Show, Don't Tell

> **不要讲"令人震撼的""不可思议的""绝美的"——让画面承担情绪。**

### 4.1 反面（Tell）

```
错误示范：
"福建的景色令人震撼，岛屿之美简直不可思议。"
"九寨沟的水是绝美的，蓝得让人窒息。"
```

### 4.2 正面（Show）

```
正确示范：
"380 万人，1400 座岛，30 年柴油发电。"
"如果我们了解九寨沟的形成历史，就会明白一处风景想要'修炼'
到九寨沟的级别，它所要经历的'磨难'将超乎想象。"
```

### 4.3 Show 的 4 个技法

1. **用数字代替形容词**："震撼" → "380 万人 / 1400 座岛"
2. **用动作代替状态**："美" → "一杯水从山顶流到山脚，要走 4 小时"
3. **用对比代替评价**："壮美" → "在你家客厅能放下的西湖，是 3000 个"
4. **用人物代替风景**："福建真美" → "海岛教师：学校的灯一直亮着"

## 5. 三连接规则（连接差异元素时必用）

> 当用户要求连接 2 个及以上差异元素时（如福建的山 + 海、北京的历史 + 现代、人 + 技术），必须建立 3 个层次的连接。

| 层次 | 功能 | 范例（福建山+海） |
|------|------|----------------|
| **技术 / 逻辑连接** | 让视频理性立得住 | 山抬升水汽形成降水，海送来水汽 |
| **物理 / 能量连接** | 让画面有动感和过程感 | 水循环（海水蒸发→山地降水→河流入海） |
| **人物 / 情感连接** | 让人记得住 | 山民 + 渔民 + 海岛教师共同的故事 |

### 5.1 三连接反模式（绝对避免）

| 反模式 | 现象 | 后果 |
|--------|------|------|
| **只有技术连接** | "X 和 Y 都用了 Z 技术" | 视频变成产品说明书 |
| **强行类比** | "X 像 Y，像 Z，像 W" | 没有真正的内在连接 |
| **虚假并列** | "X 和 Y 都是有趣的" | 没有因果链 |
| **缺失人物** | 整支视频没有人出场 | 观众无法代入 |

详见 [`three-connections.md`](./references/three-connections.md)。

## 6. 五阶段情绪弧线

```
情绪
 ↑           上扬   发展   下坠     回升        绵长
 │            ↗     ───    ↓        ↗           ↘
 │           ╱      ╱      ╲       ╱             ╲___
 │          ╱      ╱        ╲     ╱                  ╲___
 │         ╱      ╱          ╲   ╱                       ╲____
 │        ╱      ╱            ╲ ╱
 │       ╱      ╱              ╳
 │      ╱      ╱              ╱ ╲
 │     ╱      ╱              ╱   ╲
 │    ╱______╱             __╱     ╲___
 │___╱
 └────────────────────────────────────────────→ 时间
```

| 阶段 | 时长占比 | 功能 | 关键技法 |
|------|---------|------|---------|
| **上扬** | 0-15% | 让观众爱上主题 | 壮美、好奇、敬畏 |
| **发展** | 15-35% | 引入问题或冲突 | 对比、细节、共情 |
| **下坠** | 35-55% | 危机或揭示时刻 | 静默、反转、惊讶 |
| **回升** | 55-80% | 解决方案或希望 | 数据、人物行动、动能 |
| **绵长** | 80-100% | 留在观众心中的意味 | 尺度切换、诗意、回环 |

**关键约束**：
- 至少 **2 个明显高点**（上扬 + 回升）
- 至少 **1 个明显低点**（下坠）
- 结尾**不能是高点**——绵长阶段要让情绪缓慢下降
- 整体曲线**不能是平的**——平的曲线意味着没有戏剧张力

短视频（< 3 分钟）压缩为 3-beat：钩子 / 紧张 / 回报。

详见 [`emotion-arc.md`](./references/emotion-arc.md)。

## 7. 一句话 / 锚定句 / 金句

### 7.1 一句话命题（10-30 字）

定义本期视频要回答的"基本问题"。

```
模板 A: X 不是一个 Y，而是一套 Z。
模板 B: 看似矛盾的两个词，构成 XX 的全部底色。
模板 C: X 不是 Y 的工具，X 是 Y 的一部分。
```

### 7.2 锚定句（最后 30 秒，30-50 字）

定义本期视频给出的"经典理念"。可截图、可转发。

```
范例：
"微电网不是大电网的缩小版，它是海岛的独立人格。"
"一切磨难皆是修炼，一切毁灭皆是创造。"
"电不是基础设施，是命。"
```

### 7.3 金句（每 1-2 分钟 1 句）

节奏点上的"金句"——让观众愿意截图。

```
金句公式：
- 时间尺度 + 空间跨度 + 一句话定义
- 矛盾并置（1万年之古 + 100年之新）
- 拟人哲理（一切 X 皆是 Y）
- 诗意短句（凝固的江山岁月，流动的纸墨烟水）
```

## 8. 四字原则

> **星球所 / 桢公子的克制美学**

- **动词优先**：用动词不用形容词
- **名词具体**：用具体名词不用抽象名词
- **数字精确**：用精确数字不用模糊词
- **空间清晰**：用空间方位词配合画面（北上、南下、横穿、纵贯）

反面：

| 错误 | 修正 |
|------|------|
| "令人震撼的福建" | "380 万人 / 1400 座岛" |
| "非常多的山" | "山地占 90% 以上" |
| "在那个方向" | "从福州向南 300 公里" |
| "美好的画面" | 删掉，让画面自己说话 |

## 9. 钩子选择流程

> 视频的前 30 秒决定留存率。

### 9.1 选择流程

1. 把"灵魂四问"中的核心命题写出来
2. 判断选题属于哪一类（数字震撼 / 反常识 / 问题驱动 / 画面驱动 / 故事 / 议题）
3. 从对应类别中选 3 个候选钩子
4. 每个钩子都尝试往下写 100 字，看哪个最容易展开
5. 选定后再写正文，**不要中途换钩子**

### 9.2 钩子禁区（绝对不要用）

| 禁区 | 原因 |
|------|------|
| "今天我们来聊聊 XX" | 老套、零信息量 |
| "大家好，我是 XX，欢迎收看" | 占用前 5 秒宝贵钩子位 |
| "在这个信息爆炸的时代……" | 鸡汤式开场，观众立刻划走 |
| 直接抛问题但不给答案预告 | 观众没有理由继续看 |
| 钩子与正文无关 | 完播率暴跌 |

详见 [`hook-formulas.md`](./references/hook-formulas.md)（30 个公式 + 范例）。

## 10. 反模式（绝对不要做）

| 反模式 | 现象 | 后果 |
|--------|------|------|
| **没有研究就写** | 直接生成方案 | 沦为空话文案 |
| **一稿定终身** | 跳过对比矩阵直接深化 | 失去方案多样性 |
| **强行抒情** | "令人震撼""绝美""不可思议" | 油腻，观众划走 |
| **形容词堆砌** | 一段 3 个情绪词 | 没有信息量 |
| **缺失人物** | 全程只有数据和风景 | 观众无法代入 |
| **钩子与正文脱节** | 钩子提的问题正文不答 | 完播率暴跌 |
| **商务擦边** | 产品露出过于直接 | 商务项目失去信任 |
| **数据无来源** | 数字不标参考文献 | 不可信，被打脸 |
| **史实错误** | 年份 / 人物 / 事件错 | 一票否决 |
| **政治敏感** | 涉政 / 涉民族 / 涉宗教 / 涉领土 | 一票否决 |
