## 1. 城市系列（"什么是 XX" 文体）

### 1.1 《什么是武汉》
> "因江水穿城 武汉被叫做'江城'，而有了湖，武汉才有了'江湖'。东湖，便是武汉'最快意的江湖'。"
> —— 极简破题：先下定义（江城），再翻一层（江湖），最后聚焦到主角（东湖）。

### 1.2 《什么是成都》
> "3000年以来，上至王侯将相，下至升斗小民，他们逐一登临这个众生的舞台，共同营建出了中国最具人间烟火气的城市。"
> —— 时间跨度 + 阶层跨度 + 一句话定义（人间烟火气）。

### 1.3 《什么是郑州》
> "郑州既有1万年之古，几乎奠定了华夏文明的根基；郑州还有100年之新，如同中华民族伟大复兴的缩影，堪称'1万年的新城'。"
> —— 矛盾式命题："1万年之古"与"100年之新"的并置，是星球研究所最爱用的核心矛盾结构。

**写作模板（"什么是 XX" 系列）：**
```
第一段：一句话定义（看似简单）
第二段：一个反常识/惊讶的事实（让人停下来）
第三段：抛出核心命题（一句话高度概括）
第四段：分章节展开
第五段：升华与回环
```

## 2. 国家 / 文明系列

### 2.1 《中国从哪里来？》
> "身为人类，我最好奇的问题是我从哪里来？将往何处去？而身为中国人，最好奇的问题会变成中国从哪里来？将往何处去？"
> —— 哲学式钩子 + 身份切换，把"大问题"压在观众心里。

> "类似于宇宙起源于一次大爆炸，现今中国的地理格局，则与一次大碰撞息息相关。"
> —— 类比式钩子：把陌生事件映射到熟悉事件。

> "在地球最近的5亿年间，这样的板块碰撞曾发生过多次，但只有此次碰撞引发了超大幅度的地表隆起。地球上最高、最厚、最年轻的高原——青藏高原，诞生了。"
> —— 数字密度 + 极值排列 + 一句话破题。

### 2.2 《100 年，重塑山河》
> "100年来，中国做了什么？"
> —— 看似简单的问题，藏着一个宏大命题。

> "从点亮第一盏灯到实现全民通电，从第一条铁路建成通车到高铁里程位居世界第一，从发射第一颗人造卫星到登陆火星……"
> —— 排比句 + 时间跨度 + 由小到大。

### 2.3 《为 14 亿人治理江河，有多难？》
> "你见过这样的中国地图吗？"
> —— 画面式钩子：一句话勾起观众想象。

> "450亿立方米的库容，被形象喻为'可容纳3000余个西湖'。"
> —— 数字 + 类比陌生化（"3000个西湖"）。

> "从大禹治水、孙叔敖修建芍陂、李冰构筑都江堰……到苏轼修筑西湖苏堤、潘季驯以'束水攻沙'之法治理黄河……"
> —— 排比式历史回溯，从古到今的传承感。

## 3. 自然 / 地理系列

### 3.1 《三江并流有多神奇？》
> "怒江、澜沧江、金沙江在百公里内近乎平行奔流——仿佛在北京与天津间容纳三条汹涌河段，在上海与南京间矗立梅里雪山等一众5000米级高峰，在广州与深圳间汇聚中国五分之一高等植物、四分之一脊椎动物与民族，乃至半个地球的气候类型。"
> —— 这是星球研究所最经典的"类比陌生化"模板：**用中国观众熟悉的距离 + 城市 + 物种分布，重构一个陌生地理奇观**。

### 3.2 《九寨沟会消失吗？》
> "如果我们了解九寨沟的形成历史，就会明白一处风景想要'修炼'到九寨沟的级别，它所要经历的'磨难'将超乎想象。对于大自然而言，这点损伤实在微不足道，更何况，一切磨难皆是修炼，一切毁灭皆是创造。"
> —— 拟人化叙事 + 哲理升华。这是星球研究所最常用的结尾公式。

## 4. 命题公式（按用途分类）

### 4.1 钩子公式（30秒内抓住观众）

| 公式 | 范例 |
|------|------|
| 画面描述 + 极致数字 | "4.5万条江河在中国大地上奔流。" |
| 反常识事实 | "没有大碰撞，就没有今天的中国。" |
| 一句话问题 | "中国从哪里来？" |
| 类比陌生化 | "仿佛在北京与天津间容纳三条汹涌河段……" |
| 设问-答案预告 | "为什么 X？答案藏在大地之上。" |
| 身份切换 | "身为人类……而身为中国人……" |

### 4.2 立意公式（开篇定义"我们要讲什么"）

| 公式 | 范例 |
|------|------|
| 时间尺度 + 空间跨度 + 一句话定义 | "3000年以来……中国最具人间烟火气的城市。" |
| 矛盾并置 | "1万年之古 + 100年之新 = 1万年的新城。" |
| 反转式定义 | "它不是 X，而是 Y。" |
| 隐喻式命题 | "世界上根本没得四川，是大地与群山创造了一个庞大的超级'结界'。" |

### 4.3 递进公式（章节展开）

| 公式 | 范例 |
|------|------|
| 因果链 | 板块碰撞 → 青藏高原 → 风机 → 季风 → 烟雨江南 |
| 时空压缩 | 6500 万年 → 1 万年 → 100 年 |
| 自然→人文→工程 | 山川 → 文明 → 现代重塑 |
| 微观→宏观 | 一个湖泊 → 一座城市 → 一个国家 |

### 4.4 升华公式（结尾）

| 公式 | 范例 |
|------|------|
| 拟人哲理 | "一切磨难皆是修炼，一切毁灭皆是创造。" |
| 诗意短句 | "凝固的江山岁月，流动的纸墨烟水，永恒的南京底色。" |
| 回环呼应 | 回到开头画面 + 一句诗化短句 |
| 时代寄语 | "没有一个民族的崛起是命中注定的。" |

## 5. 桢公子作品专属风格提示

### 5.1 桢公子出镜的"开场白"惯例
- 通常以一个画面 / 一个场景切入；
- 不直接讲主题，而是讲"为什么我要带你看这个"；
- 语气克制，音量偏低，节奏比所长慢。

### 5.2 桢公子作品中的"主持词 / 文案"分工
- **主持词**（桢公子）：引出章节、过渡、收尾；
- **文案**（主笔 + 桢公子监制）：主体内容、画面描述、金句。
- 主笔写完后，桢公子会做"声音适配"——把不适合朗读的长句拆成短句。

### 5.3 桢公子的克制美学
- 不写"令人震撼的""不可思议的""绝对值得一看"等情绪词；
- 偏好动词、名词、数字；少用形容词；
- 让画面承担情绪，让声音承担节奏。

## 8. 三大时间尺度与三大空间维度（撰稿选题工具）

### 8.1 三大时间尺度（耿华军原话）
> "6500万年的华夏山水，1万年的文明史诗，100年的重塑山河。"

| 时间尺度 | 对应命题类型 | 范例 |
|---------|------------|------|
| **6500万年** | 自然地理、华夏山水 | 青藏高原 / 长江 / 三江并流 |
| **1万年** | 中华文明、人类活动 | 农耕起源 / 早期城市 / 文明兴衰 |
| **100年** | 重塑山河、现代工程 | 100年重塑山河 / 为14亿人治理江河 / 数字孪生水利 |

### 8.2 三大空间维度
| 维度 | 内容 |
|------|------|
| **自然地理** | 山川、河流、气候、生态 |
| **人文历史** | 文明、文化、历史、人物 |
| **现代工程** | 基础设施、产业、城市、科技 |

撰稿时优先选择 **1-2 个时间尺度 + 1-2 个空间维度** 作为主线。

| 作品 | 一句话命题 |
|------|----------|
| 什么是武汉 | 江城 → 江湖 |
| 什么是成都 | 烟火人间三千年 |
| 什么是郑州 | 1万年的新城 |
| 什么是甘肃 | ？（待研究） |
| 啥子是四川 | 结界里的天府之国 |

撰稿"什么是 XX"系列的步骤：
1. 选择目标（城市 / 省份 / 山河 / 工程）；
2. 找到它的"反常识真相"（如成都的"结界"、武汉的"江湖"）；
3. 用一句话定义 + 一句话历史 + 一句话反差的三段式破题；
4. 三大递进：地理底层 → 人文回应 → 当代重塑；
5. 收尾用诗化短句呼应收头。
