# 快速模式（Quick Mode）

> **不写完整脚本，只产出局部创意** —— 钩子、方案、金句、开头、结构等。

---

## 1. 触发条件

用户说以下任一话术，进入快速模式：

- "给我 3 个钩子"
- "给我 5 个金句"
- "给我几个方案方向"
- "我想想开头怎么写"
- "帮我列几个选题"
- "给我几个比喻"
- "我只要 30 秒的开场白"
- 任何"局部创意"请求

---

## 2. 快速模式的 4 个原则

### 2.1 仍然要问"灵魂四问"（压缩版）

不要问 4 个问题，**只问 1 个**：

```
"你想做什么样的视频？给我 1-2 句话描述，包括：
- 主题（必填）
- 时长（默认 10 分钟，可改）
- 风格（默认星球所，可改）
- 已有想法（可选）"
```

### 2.2 不做完整 6 阶段

| 完整模式 | 快速模式 |
|---------|---------|
| 灵魂四问 | 1 个压缩问题 |
| Phase 1 创作者研究 | **跳过**（默认星球所） |
| Phase 2 事实研究 | **可选**（用户没给事实时才做） |
| Phase 3 风格-事实合成 | **跳过** |
| Phase 4 3 套方案 | **跳过**（除非用户明确要方案） |
| Phase 5 深化 | **跳过** |
| Phase 6 交付 | **直接交付** |

### 2.3 直接给"可复用的产物"

不同问法的对应输出：

| 用户问 | 输出 |
|--------|------|
| "给我 3 个钩子" | 3 个钩子段落（每个 100-150 字），标注公式编号 |
| "给我 5 个金句" | 5 个金句，标注使用场景 |
| "给我几个方案方向" | 2-3 个方向（每个 1-2 句话） |
| "给我一个开头" | 1 个 200 字钩子段落 |
| "给我几个比喻" | 3-5 个类比陌生化句子 |

### 2.4 标注可升级到完整模式

每次快速模式输出末尾加：

```markdown
---

💡 **想要深入？** 选定一个方向，我可以按完整模式走完 6 阶段
（创作者研究 → 事实研究 → 3 套方案 → 深化 → 交付）。
只需说"按完整模式深化 XXX"。
```

---

## 3. 快速模式的工作流（实操）

```
用户问"给我 3 个钩子"
       ↓
[1] 问 1 个压缩问题（主题 + 时长）
       ↓
[2] 从 hook-formulas.md 选 3 个不同类型的公式
   （如：数字震撼 + 反常识 + 故事驱动）
       ↓
[3] 每个钩子写 100-150 字
       ↓
[4] 标注公式编号 + 适用风格
       ↓
[5] 加"可升级"提示
       ↓
[6] 交付
```

## 4. 快速模式的输出模板

### 4.1 钩子输出

```markdown
## 3 个钩子方案

### 钩子 A（数字震撼 + 反常识）
> 公式编号：[hook-formulas.md #1 极致数字 + 常见事物类比]
> 适用风格：星球所 / 地球知识局
>
> "在中国东南沿海，有一座省份。它有 1400 座岛屿，比海南省还多。
> 它有 380 万海岛居民。过去 30 年，他们靠柴油发电机，照亮了黑夜。"
> （约 120 字）

### 钩子 B（反常识 + 哲学式）
> 公式编号：[hook-formulas.md #6 你以为 X，其实是 Y]
> 适用风格：星球所 / 地球知识局
>
> "你以为四川是'天府之国'？其实世界上根本没有四川。"
> （约 80 字）

### 钩子 C（故事驱动 + 人物悬念）
> 公式编号：[hook-formulas.md #21 人物悬念]
> 适用风格：小约翰可汗 / 何同学
>
> "2014 年 7 月，超强台风'麦德姆'登陆福建。在福建东北角的东吾洋，
> 一座常住人口 8000 人的小岛，整整停电了 7 天。那 7 天里，海岛上的
> 小学，是全岛唯一有光的地方。"
> （约 120 字）

---

💡 想要深入？选定一个钩子，按完整模式走完 6 阶段。
```

### 4.2 金句输出

```markdown
## 5 个金句

1. "一切磨难皆是修炼，一切毁灭皆是创造。" — 拟人哲理（星球所）
2. "380 万人 / 1400 座岛 / 30 年柴油。" — 数字堆叠（克制）
3. "电不是基础设施，是命。" — 反常识定义（人物驱动）
4. "微电网不是大电网的缩小版，它是海岛的独立人格。" — 隐喻命题
5. "凝固的江山岁月，流动的纸墨烟水，永恒的南京底色。" — 诗意短句

---

💡 想要金句的完整使用场景（哪段 / 哪种风格）？按完整模式深化。
```

## 5. 什么时候应该升级到完整模式

如果用户继续追问以下任一问题，**主动建议升级**：

- "这个钩子怎么展开？"
- "我想看完整脚本"
- "拍这个需要多少钱？"
- "有没有其他方案？"
- "对比一下"

升级话术：

```
你这个问题已经超出快速模式范围了。要不要切换到完整模式？
完整模式会走完 6 阶段（创作者研究 → 事实研究 → 3 套方案 → 深化 → 交付），
给你生产就绪的脚本。只需要说"按完整模式"就行。
```
