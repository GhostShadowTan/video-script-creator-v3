# 星球研究所三道审校关 + 整体质量门

> 源自魏桢（桢公子）公开访谈："在科学性上，'作者自审、交叉审核、行业专家审核'是星球研究所推文的三道关卡。"
> 本清单用于撰稿人自查 + 团队互审 + 专家终审 + 整体交付质量门。

---

## 第一关：作者自审（撰稿人自查）

### 1.1 命题层面
- [ ] **一句话命题**是否清晰？（用一句话能讲清楚本期视频要回答的"基本问题"）
- [ ] **经典理念**是否明确？（用一句话能讲清楚本期视频给出的"经典理念"）
- [ ] **核心矛盾 / 反常识事实**是否抓人？（看完命题，普通人会想继续看吗？）
- [ ] **时间尺度**是否选定？（6500万年 / 1万年 / 100年 / 当下）
- [ ] **空间尺度**是否选定？（全国 / 区域 / 一城一山一河）

### 1.2 骨架层面
- [ ] 是否有**钩子**？（0:00-0:30）
- [ ] 是否有**立意**？（0:30-1:30）
- [ ] 是否有**至少 3 道递进**？（每一道递进都是一个章节）
- [ ] 递进之间是否有**因果链**？（不是并列，而是层层推进）
- [ ] 是否有**回环与升华**？（结尾是否呼应开头）
- [ ] 总字数 / 总时长是否在合理范围？（10分钟视频约 2500-3500 字）

### 1.3 表达层面
- [ ] 是否每 1-2 分钟出现一句**金句**？（金句密度）
- [ ] 是否每个章节开头都有**一句话命题短句**？
- [ ] 是否每 200-300 字都有一个**画面钩子**？（让导演 / 摄影师知道配什么画面）
- [ ] 是否避免了**形容词堆砌**？（"震撼的""壮美的""绝美的"等情绪词每段不超过 1 次）
- [ ] 是否做到了**长短句交错**？（给桢公子的声音以"呼吸点"）
- [ ] 是否使用了**空间方位词**？（北上、南下、横穿、纵贯 — 配合航拍画面）

### 1.4 数据 / 史实层面
- [ ] 每个数字是否有**来源**？（精确到文献 / 论文 / 报告 / 官方网站）
- [ ] 数字是否**单位一致**？（避免混用 km / 千米 / 公里）
- [ ] 数字是否**精确**？（避免"大约""差不多""左右"等模糊词）
- [ ] 类比是否**准确**？（"相当于 3000 个西湖"是否真的等价）
- [ ] 史实是否有**时间坐标**？（朝代、年份、世纪）

## 第二关：交叉审核（团队互审）

### 2.1 团队成员（含所长 / 桢公子 / 导演 / 动画师）分别检查：

**所长视角（耿华军）：**
- [ ] 这期视频是否回答了一个"基本问题"？
- [ ] 是否提出了一个"经典理念"？
- [ ] 是否经得起"三年 / 五年 / 十年"的考验？

**桢公子视角（魏桢）：**
- [ ] 念出来是否**流畅**？（拗口的长句要拆短）
- [ ] 是否每句话都有**画面**？（念这句时，画面上是什么？）
- [ ] 整体气质是否**克制**？（没有过度抒情）

**导演视角：**
- [ ] 每段文案是否有**画面钩子**？
- [ ] 节奏是否**有张有弛**？（避免每段都密）
- [ ] 是否预留了**呼吸点**？（让画面独立说话）

**动画师视角：**
- [ ] 抽象概念是否有**可视化方案**？
- [ ] 数字是否有**视觉表达**？（柱状图 / 比例图 / 地图）
- [ ] 时空跨度是否有**动效**？（时间线 / 地图演变）

**地图师视角：**
- [ ] 是否需要新增地图？
- [ ] 地图上的标注是否清晰？
- [ ] 空间方位是否准确？

**素材编辑视角：**
- [ ] 是否列出所有需要的素材（航拍 / 地面 / 历史图片 / 3D）？
- [ ] 是否有素材缺口？（如果缺，能否补拍或用 3D 替代？）

### 2.2 团队互审会输出三份反馈：
1. **必须改**：影响命题 / 史实 / 节奏的问题；
2. **建议改**：可优化但不影响主线的问题；
3. **可保留**：暂时不改，看后续反馈。

## 第三关：行业专家审（终审）

### 3.1 邀请对应的行业专家

| 选题类型 | 推荐专家 |
|---------|---------|
| 自然地理 / 地质 | 中科院 / 高校地质 / 地理学者 |
| 历史人文 | 高校历史学者 / 博物馆研究员 |
| 工程 | 行业内部专家 / 设计院工程师 |
| 生态 / 生物 | 自然保护区研究员 / 高校生态学者 |
| 城市 | 城市规划学者 / 当地文史专家 |

### 3.2 专家审稿输出模板：
```
【专家审稿意见 - XX 主题】

史实性问题：
1. （如有，请具体指出，并提供权威来源）
2. ……

数据性问题：
1. ……

观点性问题：
1. ……

建议性意见（可改可不改）：
1. ……

总体评价：
A. 可发布
B. 需小修
C. 需大修
```

## 4. 一票否决项（出现任何一条，必须修改）

| 编号 | 一票否决项 |
|------|-----------|
| 1 | 核心数据错误（年份 / 海拔 / 距离 / 数量） |
| 2 | 史实错误（事件 / 人物 / 时间） |
| 3 | 抄袭 / 洗稿（未注明出处的原文复用超过 50 字） |
| 4 | 政治敏感（涉政 / 涉民族 / 涉宗教 / 涉领土） |
| 5 | 价值观偏离（如性别歧视、地域歧视等） |
| 6 | 缺乏科学依据的伪科学 |
| 7 | 立意不清晰（讲完一遍不知道在讲什么） |
| 8 | 画面与文字脱节（每分钟超过 3 句无画面） |
| 9 | **商务项目专项**：产品露出过于直接（违反"人文化包装"原则） |
| 10 | **星球所专项**：钩子与正文无关 |

---

## 5. 加分项（出现越多越好）

- 金句密度高于平均水平；
- 类比陌生化精准（让人"哇"出来）；
- 时间跨度和空间跨度都拉满；
- 结尾升华与开头呼应；
- 让读者产生"转发冲动"（态度大于信息）；
- 文字可以脱离视频独立成稿（长期留存）；
- **商务项目专项**：产品是"自然发现的"而非"被推销的"；
- **星球所专项**：3 道递进有清晰的因果链。

## 6. 整体质量门（6 阶段交付前自检）

### Phase 1-2 完成标志
- [ ] ≥5 位创作者分析，具体技法级洞察
- [ ] ≥15 个事实元素，有可追溯来源

### Phase 3-4 完成标志
- [ ] 3 个方案彼此清晰可区分
- [ ] 3 个方案各有 3 个金句候选
- [ ] 对比矩阵含通用 + 商务专项 + 星球所特色维度

### Phase 5 完成标志
- [ ] 选定方案有五阶段情绪弧线
- [ ] "三连接"规则满足（如连接差异元素）
- [ ] 最后 30 秒至少 1 个锚定句
- [ ] 生产指导包含拍摄 / 音乐 / 旁白 / 视觉

### Phase 6 交付标志
- [ ] 所有来源带可点击链接引用
- [ ] 走 `html-report` / `docx` skill 交付
- [ ] **星球所专项三关清单全部通过**

## 7. 星球研究所的"长期主义"标准（终极标准）

> 魏桢原话："我们创作的核心原则是解答基本问题，提出经典理念。让内容能够经得住时间的考验。"
> "我们距离百年老店，还有 90 年。"

**终极自检问题：**
> 三年后、五年后、十年后再看这期视频，它还会让人想看第二遍吗？
> 如果答案是"会"——通过。
> 如果答案是"会，但需要更新数据"——通过。
> 如果答案是"热点过了就没意义了"——重做。
