# 五阶段情绪弧线模型（Emotion Arc）

> 视频不是事实的堆砌，而是**情绪的设计**。每支脚本必须有意识地设计观众的情绪曲线，否则再多的素材也只是流水账。
> 本模型是星球研究所 / 何同学 / BBC 纪录片共通的"心理工程学"。

---

## 1. 五阶段总览

| 阶段 | 时长占比 | 功能 | 关键技法 |
|------|---------|------|---------|
| **上扬** | 0-15% | 让观众爱上主题 | 壮美、好奇、敬畏 |
| **发展** | 15-35% | 引入问题或冲突 | 对比、细节、共情 |
| **下坠** | 35-55% | 危机或揭示时刻 | 静默、反转、惊讶 |
| **回升** | 55-80% | 解决方案或希望 | 数据、人物行动、动能 |
| **绵长** | 80-100% | 留在观众心中的意味 | 尺度切换、诗意、回环开头画面 |

---

## 2. 每个阶段的详细指导

### 2.1 上扬（Rising）
- **目标**：让观众爱上主题 / 想要继续看
- **技法**：
  - 壮美画面（航拍、延时、特写）
  - 极致数字（"3000 个西湖"）
  - 身份切换（"身为中国人……"）
  - 类比陌生化（"仿佛在北京与天津间容纳三条大江"）
- **星球研究所范例**：
  > "你见过这样的中国地图吗？"（《为14亿人治理江河》）
- **时长**：30 秒 - 1 分 30 秒
- **质量门**：30 秒留存率 > 70%

### 2.2 发展（Developing）
- **目标**：建立背景，引入张力
- **技法**：
  - 反差对比（"今天 vs 昨天"）
  - 具体细节（一个人、一栋楼、一棵树）
  - 共情锚点（人物故事、场景）
  - 知识铺垫（但不超过 2 分钟）
- **星球研究所范例**：
  > "在没有三峡水库之前，1998 年那场洪水让沿江民众身陷险境……"（《为14亿人治理江河》）
- **时长**：2-3 分钟
- **质量门**：观众开始对"问题"产生好奇

### 2.3 下坠（Falling）
- **目标**：揭示真相 / 加深危机 / 出现反转
- **技法**：
  - 静默时刻（让画面说话）
  - 反转（你以为 X，其实是 Y）
  - 惊讶（数字震撼、悖论呈现）
  - 矛盾激化（把张力推到最高）
- **星球研究所范例**：
  > "如果我们了解九寨沟的形成历史，就会明白……一切磨难皆是修炼，一切毁灭皆是创造。"（《九寨沟会消失吗》）
- **时长**：2-3 分钟
- **质量门**：观众产生"原来如此"的认知冲击

### 2.4 回升（Rising Again）
- **目标**：给出答案 / 提供希望 / 展现人的能动性
- **技法**：
  - 解决方案（技术、工程、政策）
  - 数据支撑（清洁能源占比 60%）
  - 人物行动（"校长决定做一件事"）
  - 动能感（"现在正在发生……"）
- **星球研究所范例**：
  > "三江源国家公园的建立、黄河三角洲自然保护区……皆为此证。"（《为14亿人治理江河》）
- **时长**：2-3 分钟
- **质量门**：观众感到"未来可期"

### 2.5 绵长（Lasting）
- **目标**：让观众在视频结束后仍记得这 10 分钟
- **技法**：
  - 尺度切换（大→小 或 小→大）
  - 诗意短句（"凝固的江山岁月，流动的纸墨烟水"）
  - 回环开头画面（"380 万人 / 1400 座岛"）
  - 一句"可引用"的锚定句
- **星球研究所范例**：
  > "一切磨难皆是修炼，一切毁灭皆是创造。"
- **时长**：1-2 分钟
- **质量门**：观众愿意"截图 / 转发"

## 3. 情绪曲线示意（折线图）

```
情绪
 ↑
 │           上扬   发展   下坠     回升        绵长
 │            ↗     ───    ↓        ↗           ↘
 │           ╱      ╱      ╲       ╱             ╲___
 │          ╱      ╱        ╲     ╱                  ╲___
 │         ╱      ╱          ╲   ╱                       ╲____
 │        ╱      ╱            ╲ ╱
 │       ╱      ╱              ╳
 │      ╱      ╱              ╱ ╲
 │     ╱      ╱              ╱   ╲
 │    ╱______╱             __╱     ╲___
 │___╱
 └────────────────────────────────────────────→ 时间
   0   1min  2min  3min  5min  7min   9min  10min
```

**关键特征**：
- 至少 **2 个明显高点**（上扬 + 回升）；
- 至少 **1 个明显低点**（下坠）；
- 结尾**不能是高点**——绵长阶段要让情绪缓慢下降，留下余韵；
- 整体曲线**不能是平的**——平的曲线意味着没有戏剧张力。

## 4. 短视频压缩版（< 3 分钟）

对于短视频（抖音、Reels、B站短视频），把五阶段压缩为 **3-beat**：

| Beat | 时长 | 内容 | 对应五阶段 |
|------|------|------|----------|
| **钩子** | 0-5s | 反常识画面 + 一句话命题 | 上扬 + 发展 |
| **紧张** | 5s-2min | 问题 + 危机 + 关键事实 | 下坠 |
| **回报** | 2min-3min | 解决方案 + 金句 + 收尾 | 回升 + 绵长 |

短视频版的情绪曲线更陡峭，更短促。

---

## 5. 情绪弧线设计的常见错误

| 错误 | 现象 | 修正 |
|------|------|------|
| **全程上扬** | 一直高潮，没有对比，观众疲劳 | 必须在 2/3 处安排"下坠" |
| **全程下坠** | 太沉重，看完抑郁 | 必须有"回升"和"绵长" |
| **情绪断裂** | 上一秒沉重，下一秒突然轻松 | 用过渡句 / 音乐 / 画面渐变连接 |
| **结尾高潮** | 结尾仍在大声呼喊 | 结尾必须让情绪缓慢下降 |
| **共情缺失** | 只有数据，没有人 | 必须有"人物行动 / 感官细节" |
