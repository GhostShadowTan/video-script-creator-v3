# 10 位头部 UP 主笔法速查卡

> 本卡片用于撰稿时快速匹配笔法。每位 UP 主的笔法提炼为 5 个维度：**定位 / 钩子 / 主线 / 金句 / 风险**。

---

## 地理 / 自然 类

### 1. 🥇 星球研究所 — 地理科普之王

| 维度 | 内容 |
|------|------|
| **定位** | 以地理视角看中国 / 世界，重新发现脚下这片土地 |
| **钩子** | 画面描述 + 极致数字 + 类比陌生化 |
| **主线** | 三大时间尺度（6500万年/1万年/100年）+ 三大空间维度 |
| **金句** | "一切磨难皆是修炼，一切毁灭皆是创造" |
| **风险** | 学术感过重、画面与文字脱节、节奏过密 |

**代表作**：《中国从哪里来？》《什么是成都》《什么是武汉》《100年，重塑山河》《为14亿人治理江河》《三江并流有多神奇？》

**速用句式**：
- "X 不是一个 Y，而是一套 Z。"
- "看似矛盾的两个词，构成 XX 的全部底色。"
- "在 X 公里内，容纳了……"

### 2. 🥈 地球知识局 — 全球人文地理万花筒

| 维度 | 内容 |
|------|------|
| **定位** | 全球视角下的人文地理议题 |
| **钩子** | 反常识事实 + 国际对比 |
| **主线** | 议题驱动 + 国别比较 + 落到中国 |
| **金句** | "你以为 X，那就错了。" |
| **风险** | 议题选择不当易引起争议，国际比较易失客观 |

**代表作**：《俄罗斯为什么没有出海口？》《印度为什么总和中国比？》

**速用句式**：
- "如果你以为 X，那是因为你不知道 Y。"
- "同样是 Z，为什么 X 国 / Y 国 / 中国 走向了不同的命运？"

| **钩子** | 画面描述 + 空间方位 |

## 人文 / 历史 / 知识 类

### 5. 🥇 罗翔说刑法 — 法理相声

| 维度 | 内容 |
|------|------|
| **定位** | 用段子讲严肃法理 |
| **钩子** | "同学们好啊" + 一个通俗案例 |
| **主线** | 案例 → 法理 → 人性 → 反思 |
| **金句** | "法律是对人最低的道德要求。" |
| **风险** | 段子过密易消解严肃性、案例选择不当易引争议 |

**代表作**：《张三的奇妙冒险》《刑法学讲义》

**速用句式**：
- "今天张三又来了。"
- "如果 X，那 Y 是不是 Z？"
- "看似是 X 的问题，其实是 Y。"

## 科技 / 数码 / 硬核科普类

### 8. 🥇 老师好我叫何同学 — 理科生的浪漫

| 维度 | 内容 |
|------|------|
| **定位** | 用科技产品讲人和世界的关系 |
| **钩子** | "做这个视频的起因是……" |
| **主线** | 产品 → 技术 → 人 → 情感 |
| **金句** | "科技的本质，是让人和世界重新连接。" |
| **风险** | 制作成本高、依赖真实经历、节奏易拖沓 |

**代表作**：《5G 到底有多快》《我用 36 万行备忘录做了个动画》《毕业视频》

**速用句式**：
- "我做这个视频，是因为……"
- "X 不是 Y 的工具，X 是 Y 的一部分。"
- "技术参数背后，是人和人的关系。"

## 终极提示

撰稿时，先选定 **1-2 位 UP 主**作为笔法主参考，然后从星球研究所风格中提取**克制的底色**。这样既有辨识度，又不会脱离星球研究所 / 桢公子的整体气质。
